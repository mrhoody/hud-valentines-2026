### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it something like `valentines-2026` (the name will be part of your URL)
3. Keep it public
4. Don't initialize with README (we already have files)

### Step 2: Push to GitHub

Run these commands in your terminal:

```bash
git init
git add .
git commit -m "Add Valentine's Day page"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username and `YOUR_REPO_NAME` with your repository name.

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on "Settings"
3. Scroll down to "Pages" in the left sidebar
4. Under "Source", select "main" branch
5. Click "Save"

Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
